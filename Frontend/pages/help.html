Help & User Guide

DMS Platform — Document Management System System

🔄 How the Workflow Works
  1. Create Note — Initiator creates a note with subject, type, and uploads the main PDF.
  2. Select Recommenders — Add one or more recommenders (Stage 1, 2, 3…). They review sequentially.
  3. Select Approver — Add the final approving authority (DGM, AGM, etc.).
  4. Submit — Note goes to Stage 1 Recommender for check.
  5. Recommender Review — At each stage: if changes needed → returned to uploader. If OK → forwarded to next stage.
  6. Final Approval — After all recommenders clear, goes to Approver. Approver can approve or return.
  7. Note Approved — Status changes to Approved. All parties notified by email.
Note: If a note is returned from Stage 3, the uploader can resubmit after making changes — they can choose to re-enter at Stage 1, 2, or 3.

❓ FAQ

Q: Can I add multiple recommenders?
Yes. There is no limit on the number of recommenders. They will review the note sequentially.

Q: What happens when a note is returned?
You (the initiator) receive an email notification. You can view the comments, make changes, and resubmit.

Q: Can I edit a note after submission?
Only if the note has been returned to you. You cannot edit a note that is currently under review.

Q: What file formats are supported?
Main Note: PDF only. Annexures: PDF, JPG, PNG, XLSX, DOCX.

Q: How do I track the status of my note?
Go to Dashboard → All Notes. The Status column shows current state. Click any row to see the full Audit Log.
